## Варіант 3: Kanban Board (Дошка задач)

### Реалізований функціонал:
1. **Інтерфейс дошки:**
   - Три колонки: "Треба зробити", "В процесі", "Готово".
   - Динамічні лічильники задач у кожній колонці.
2. **Управління задачами (CRUD):**
   - Додавання нових задач через модальне вікно (Заголовок, опис, пріоритет).
   - Відображення кольорових маркерів залежно від пріоритету (низький, середній, високий).
   - Видалення задач по кліку на хрестик.
   - Редагування існуючих задач по подвійному кліку на картку.
3. **Drag & Drop:**
   - Повноцінне перетягування карток між колонками за допомогою HTML5 Drag and Drop API.
   - Візуальний фідбек при перетягуванні (напівпрозорість картки, зміна фону колонки).
4. **Збереження даних:**
   - Усі задачі та їхні статуси автоматично зберігаються у `LocalStorage`. При оновленні сторінки нічого не втрачається.

### Як запустити:
Проект написаний на Vanilla JS і не потребує додаткових бібліотек. Перейдіть за посиланням на GitHub Pages, щоб переглянути живу версію.
